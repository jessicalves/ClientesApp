App de Cadastro de Clientes (.NET MAUI - Windows)
Aplicativo feito em .NET MAUI para rodar no Windows, para o gerenciamento de clientes.

 Funcionalidades:
 - Cadastrar novo cliente.
 - Editar cliente.
 - Excluir cliente.
 - Listar clientes.
